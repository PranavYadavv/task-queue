# task-queue

Lightweight async task queue with retry and dead letter support

## Installation

```bash
pip install task-queue
```

## Quick Start

```python
from task_queue import Task
client = Task()
client.run()
```

## Features

- High performance and memory efficient
- Comprehensive error handling
- Well tested with full coverage
- MIT licensed
